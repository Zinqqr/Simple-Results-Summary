## Results Summary – Frontend Mentor Challenge

## Short Description
This project is a solution to the Results Summary Component challenge on Frontend Mentor. The goal was to create a responsive component that displays a user’s performance score alongside category breakdowns, using the provided design and assets.

## Features
Responsive design optimized for mobile and desktop screens.

Semantic HTML5 structure for accessibility.

Flexbox and CSS Grid for modern layout control.

Smooth hover/focus states for interactivity.

Typography and colors that match the provided style guide.

## Technologies Used
HTML5

CSS3 (Flexbox & Grid)

Google Fonts

Frontend Mentor design assets
